# UIElementsUtility

Package ID: `com.vladislavtsurikov.uielementsutility`

## Why it exists
UIElementsUtility provides reusable Unity UI Toolkit helpers and supporting editor/runtime utilities.

## Technical overview
The package includes runtime utilities, editor helpers, and content used to build UI Toolkit based tools.

This package currently contains: Runtime, Editor, Content.

## How to use
Add the package to a Unity project as a local folder or UPM Git dependency, reference `VladislavTsurikov.UIElementsUtility`, and use the helper APIs in UI Toolkit code.
