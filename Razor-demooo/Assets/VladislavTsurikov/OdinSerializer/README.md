# OdinSerializer

Package ID: `com.vladislavtsurikov.odinserializer`

## Why it exists
OdinSerializer provides serialization support used by Unity packages in this workspace.

## Technical overview
The package contains serializer core code, Unity integration, configuration, utilities, and project properties.

This package currently contains: Core, Config, Utilities, Unity Integration.

## How to use
Add the package to a Unity project as a local folder or UPM Git dependency, reference `VladislavTsurikov.OdinSerializer`, and use the serializer integration from dependent packages.
