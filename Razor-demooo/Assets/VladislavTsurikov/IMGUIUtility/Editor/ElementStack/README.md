# ElementStack

Assembly: `VladislavTsurikov.IMGUIUtility.CustomInspectorIntegration`

## Why it exists
ElementStack contains IMGUI editor helpers for stack-based custom inspector UI.

## Technical overview
The package provides editor-only components for element stacks, tab stacks, icon stacks, reorderable lists, and single-element stack inspectors.

This package currently contains: Editor.

## How to use
Add the repository to a Unity project, reference the asmdef from editor tooling that needs custom inspector integration, and use the provided stack editors and drawer helpers.
